**World Wide Web** is a distributed, online information storage space. everything is **resource**, and is identified uniquely by **URL**(uniform resource locator), and is transmitted to user by **HTTP**(hyper text transfer protocol).

1. **URL**
	```
	<protocol>://<host>:<port>/<path>
	
	<protocol>: http, https, ftp...
	<host>: the DN or ip of the host restored the resource.
	<port>, <path> usually omitted.(http use port 80, root path.)
	```
