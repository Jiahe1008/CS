# 总线

分时性，共享性。同一时刻，只允许一个设备向总线发送信号，允许多个设备从总线上读取信号。
## 分类
1. 内部总线 （片内总线）
	芯片内部的总线
2. 系统总线
	1. 数据总线
	2. 地址总线：单向总线，从cpu到存储设备。
	3. 控制总线
	4. I/O总线
3. 通信总线
	与外部I/O设备或者不同计算机系统之间的通信。
## 结构
单，双，三总线结构（分别是系统总线，存储总线，DMA）
分层总线: 南北桥芯片。

## 总线性能指标

1. 总线宽度：位数
2. 带宽： 每秒传输的字节数 = 宽度 \* 时钟频率 / 完成一次传输需要的时钟周期（工作频率）
3. 负载
4. 复用
5. 突发（猝发）传输

## 总线事务
主设备与从设备完成一次完整的信息交换过程。发出事务请求的是主设备。
