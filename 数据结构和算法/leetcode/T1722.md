# 执行交换操作之后的最小汉明距离

## CoT
1. 使用[[并查集]]  
	先通过union遍历`allowedSwaps`找到所有下标的father
2. 获得每个资源池，每个资源池内部可以任意换  
3. 则统计target和source在每个资源池中数字出现的频次  
	使用[[哈希表]]？
	使用一个数组，元素为每个资源池中各个数字出现的频次，也就是存储一个哈希表。
	```cpp
	vector<unordered_map<int, int>> pool_list(n); //pool_list[i]是一个哈希表
	// n是所有下标的个数
	```
	怎么填入这个vector？
	```cpp
	//填入过程
	for(int i=0; i<n; i++) {
		int root = uf.find(i); // find the father of the i which is also an index
		pool[root][source[i]]++;
	}
	```
4. 对每个资源池中数字和频次取交集  
5. 多个资源池的并集和相加，用总长度减去这个和就是最小汉明距离
```cpp
int ans=n;
for(int i=0; i<n; i++) {
	int root = uf.find(i);
	if(pool[root].count(target[i]) && pool[root][target[i]] > 0) 
		ans--, pool[root][target[i]]--;
}
```
