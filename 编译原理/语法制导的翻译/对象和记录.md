| 对比        | 记录 record  | 对象 object                    |
| --------- | ---------- | ---------------------------- |
| 主要内容      | 字段 / 属性    | 字段 / 属性 + 方法                 |
| 关注点       | 数据组织       | 数据和行为封装                      |
| 是否强调封装    | 一般不强调      | 强调                           |
| 是否支持继承、多态 | 一般不支持      | 通常支持                         |
| 典型例子      | C 的 struct | Java / C++ / Python 的 object |
