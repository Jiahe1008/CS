
使用具有适当数量的对象来实现一颗语法树的各个节点。
	每个对象有一个op字段，表示这个节点的标号(类型)
	*一棵树有很多节点，每个节点可以用一个对象表示。每个对象里都有一个字段叫 `op`，表示这个节点是什么类型。*
	其他字段：
		如果节点是一个叶子，那么对象将有一个附加的域，存放叶子节点的词法值。
		如果是内部节点，附加字段个数和此节点子节点个数相同。
		$Node(op, c_1, c_2, c_3 ,\dots, c_k)$

### 使用 [[L属性定义]]

| 产生式         | 语义规则                                                        |
| ----------- | ----------------------------------------------------------- |
| 1. E->TE'   | E.node = E'.syn<br>E'.inh = T.node                          |
| 2.E'->+TE1' | E1'.inh = new Node('+', E'.inh, T.node)<br>E'.syn = E'1.syn |
| 3.E'->-TE1' | E1'.inh = new Node('-', E'.inh, T.node)<br>E'.syn = E'1.syn |
| 4.E'->e     | E'.syn = E'.inh                                             |
| 5. T->(E)   | T.node = E.node                                             |
| 6. T->id    | T.node = new leaf                                           |
| 7. T->num   | T.node = new leaf                                           |
